## 🚀 Quick Start / 快速开始

1. Make sure Windows Live Captions is **enabled** / 确保Windows实时字幕已**启用**
2. Run `SaveLiveCaptions.exe` / 运行 `SaveLiveCaptions.exe`
3. Click **🔴 Start** to begin recording / 点击 **🔴 开始** 开始录制
4. Use **⏸ Pause** and **▶ Resume** for segmented recording / 使用 **⏸ 暂停** 和 **▶ 恢复** 进行分段录制

For detailed instructions, please select your preferred language above.
